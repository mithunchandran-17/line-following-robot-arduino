# Line Following Robot using Arduino

An autonomous robot designed using Arduino and IR sensors to follow a predefined path.

## Features
- Real-time line detection
- Autonomous navigation
- Motor driver integration
- Sensor-based movement control

## Components Used
- Arduino UNO
- IR Sensors
- L293D Motor Driver
- DC Motors
- Robot Chassis

## Skills Used
- Embedded Systems
- Arduino Programming
- Circuit Design
- Sensor Integration
- Motor Control
